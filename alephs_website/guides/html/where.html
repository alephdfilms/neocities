Where do I get my own website?

There's a couple of places to get a website for free. Here's a few that I've tried:

  • Neocities (www.neocities.org) - The place where this website is hosted now. If you're JUST starting out, then this is the place for you to learn how to make websites. Doesn't support some file formats though.
    This is what we'll be using to make our very own website.
  • Github Pages (pages.github.com) - If you're viewing this in Safari 2—or even Internet Explorer—blame this service. It also supports almost every file format that I can think of, so if you like using Flash files, this is the place.
  • Glitch (glitch.com) - Where I started off. No, really.
