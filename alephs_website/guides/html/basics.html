<h1>HTML Basics<h1>

Just a reminder—almost all tags need to be structured like this: <tag</tag>
However, some tags, like the <img> tag, don't need such a structure, and can be written like <this> or <this />.

Setting up your HTML file

Before continuing, always remember set your file up like this, then add whatever you need to it. 

        <!DOCTYPE html>
        <html>
        <head>
        <title>The title of your HTML page as it shows up in the browser</title>
        </head>
        <body>
        Replace this text with whatever code you want here.
        </body>
        </html>

Text Tags

<p> - The paragraph tag.
Have you ever wanted to write an essay in HTML? Well now you can. 

	Here's an example on how to code a paragraph in HTML:
	<p>This is a paragraph!</p>
	<p>You can't nest them, but you can break them apart!</p>
And here it is when viewed in a browser:

This is a paragraph!

You can't nest them, but you can break them apart!

Simple, right? Right! Paragraphs should be used to separate walls of text without needing the <br> tag.
